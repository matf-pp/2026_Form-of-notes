## About
A GUI productivity app for managing small notes, tasks and calendar.

## Tools and technologies
- [Rust](https://rust-lang.org/)
- [Cargo](https://crates.io/)
- [Slint](https://slint.dev/)

## Building the app
Clone the repository:
```
git clone https://github.com/matf-pp/2026_Form-of-notes.git
cd 2026_Form-of-notes
```

Build the app:
```
cargo build
```

Run the app:
```
cargo run
```

## Running the app
The project was developed for Linux and Windows operating systems.

Download and extract the executable for your operating system from  [Releases](https://github.com/matf-pp/2026_Form-of-notes/releases/latest).

Run the app:
```
#Linux
./form_of_notes
```

```
#Windows
form_of_notes.exe
```
